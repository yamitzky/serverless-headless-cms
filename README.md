## Development

Create .env file.

```bash
NEXT_PUBLIC_FIREBASE_CONFIG={"apiKey":"AIzaSy..."}
```

```bash
yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
